## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/react-js-personal-portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   cd react-js-personal-portfolio
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm start
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
